# RadWare

**64 服务器上安装了 RadWare 5.3 程序**

## 配置

在个人目录下的 *.bashrc* 文件中添加以下配置

```bash
source /opt/RadWare053/.radware.bashrc
export PATH=$PATH:/opt/RadWare053/bin
```
